# 📚 دليل طباعة الدفعات - Batch Printing

## ما هي ميزة الدفعات؟

ميزة تسمح لك برفع **عدة صور من البطاقات مرة واحدة** وطباعتها جميعها على **ورقة A4 واحدة** بشكل منسق وسهل!

---

## 🚀 كيفية الاستخدام

### الخطوة 1️⃣: اذهب إلى تبويب الدفعة
```
اضغط على التبويب: "📚 دفعة"
```

### الخطوة 2️⃣: رفع الصور الأمامية
```
1. انقر على "اختر عدة صور أمامية..."
2. اختر جميع صور البطاقات الأمامية مرة واحدة
   (استخدم Ctrl+Click لاختيار متعدد)
3. يمكنك اختيار 1 صورة أو 100 صورة!
```

### الخطوة 3️⃣: رفع الصور الخلفية (اختياري)
```
إذا كان لديك صور خلفية:
1. انقر على "اختر عدة صور خلفية..."
2. اختر صور البطاقات الخلفية
3. ترتيبها يجب أن يطابق الصور الأمامية

ملاحظة: إذا لم تحمل صور خلفية، ستتم طباعة الجانب الأمامي فقط
```

### الخطوة 4️⃣: اختر عدد البطاقات على الصفحة
```
الخيارات:
- 4 بطاقات (2×2)
- 6 بطاقات (2×3)
- 8 بطاقات (2×4)
- 10 بطاقات (افتراضي)
- مخصص: اختر عدد الصفوف والأعمدة حسب رغبتك
```

### الخطوة 5️⃣: اطبع أو صدّر
```
اختر:
- "طباعة الدفعة 🖨️": للطباعة مباشرة
- "تصدير PDF للدفعة 📄": لحفظ ملف PDF
```

```
✓ رفع 20 صورة أمامية
✓ اختيار 10 بطاقات لكل صفحة
✓ النتيجة: صفحتان A4 (10 بطاقات لكل صفحة)

✓ رفع 15 صورة أمامية

✓ اختيار 6 بطاقات (2×3)
✓ النتيجة: 3 صفحات A4 (6 بطاقات لكل صفحة)
```

### مثال 3: ترتيب مخصص
```
✓ رفع 12 صورة
✓ اختيار "مخصص"
✓ إدخال: 3 صفوف × 4 أعمدة
✓ النتيجة: صفحة واحدة بـ 12 بطاقة بترتيب 3×4
```

## 🎯 ملاحظات مهمة

### ترتيب الصور
```
⚠️ الصور ستُطبع بالترتيب الذي تم رفعها به
✓ الصورة الأولى المرفوعة = البطاقة الأولى المطبوعة
✓ الصورة الثانية = البطاقة الثانية، وهكذا...
```

### الترتيب على الصفحة
```
مثال: إذا اخترت 2×3 (صفين × 3 أعمدة):

  [1] [2] [3]
  [4] [5] [6]

البطاقة 1 في أعلى يسار
البطاقة 2 في منتصف أعلى
... وهكذا (من اليسار إلى اليمين ثم الصف التالي)
```

### المحاذاة
```
✓ البطاقات تُوضع تلقائياً بشكل متساوٍ
✓ المسافات بين البطاقات محسوبة بدقة
✓ الهوامش محفوظة تلقائياً
```

## 💡 نصائح عملية

### 1. إدارة الصور
```
✓ احفظ صورك في مجلد واحد لسهولة الاختيار
✓ رتب الصور باسماء منطقية (Card_1, Card_2, إلخ)
✓ استخدم نفس دقة الصور جميعها
```

### 2. الطباعة المثالية
```
✓ اختبر على ورقة عادية أولاً
✓ تحقق من المحاذاة والفراغات
✓ اطبع على ورق عالي الجودة (280+ gsm)
✓ استخدم إعدادات طابعتك الأفضل
```

### 3. عند التصدير إلى PDF
```
✓ حفظ الملف بمكان آمن
✓ تحقق من أن جميع الصور في PDF
✓ يمكنك فتح PDF وتعديله قبل الطباعة
✓ شارك PDF مع مطبعة احترافية إذا أردت
```

## ⚙️ الإعدادات المتقدمة

### تغيير عدد البطاقات
```
حالة المرونة العالية:
- 4 بطاقات: توفير ورق، نسخة محدودة
- 6 بطاقات: توازن جيد
- 10 بطاقات: الحد الأقصى العملي
- مخصص: حسب احتياجاتك الخاصة
```

### تخطيط مخصص
```
أمثلة:
- 1×1: بطاقة واحدة لكل صفحة (نادراً)
- 2×5: 10 بطاقات في صف واحد
- 5×2: 10 بطاقات في عمود واحد
- 4×3: 12 بطاقة (3 صفوف × 4 أعمدة)
```

## 🖨️ إعدادات الطابعة

### للطباعة من الموقع:
```
1. اضغط "طباعة الدفعة"
2. في نافذة الطباعة اختر:
   ✓ حجم الورقة: A4
   ✓ الاتجاه: عمودي (Portrait)
   ✓ الهوامش: None (بدون هوامش)
   ✓ الجودة: Best Quality
3. اضغط "Print"
```

### للطباعة المزدوجة الوجه (إذا أردت):
```
ملاحظة: طباعة الدفعات عادةً تكون للجانب الأمامي
لكن إذا أردت:
1. اطبع الجانب الأمامي أولاً
2. ثم ادخل الورق مجدداً
3. اطبع الجانب الخلفي يدوياً
```

## ❌ حل المشاكل الشائعة

### المشكلة: الصور لا تظهر في المعاينة
```
الحل:
1. تأكد من أن الصور بصيغ مدعومة (JPG, PNG)
2. جرب إعادة تحديث الصفحة (F5)
3. امسح ذاكرة التخزين المؤقت
```

### المشكلة: البطاقات غير محاذاة على الصفحة
```
الحل:
1. تأكد من اختيار "بدون هوامش" في الطابعة
2. جرب ترتيب مختلف (4 أو 6 بدلاً من 10)
3. اطبع نموذج اختبار قبل الطباعة النهائية
```

### المشكلة: PDF كبير الحجم جداً
```
الحل:
1. ضغط الصور قبل رفعها (استخدم TinyPNG)
2. استخدم صيغة JPEG بدلاً من PNG
3. قلل عدد البطاقات لكل صفحة
```

### المشكلة: الترتيب خاطئ
```
الحل:
1. تحقق من ترتيب الصور المرفوعة
2. الصور تُطبع بالترتيب الذي تختارها
3. استخدم معايِر واضحة (Card_01, Card_02, إلخ)
```

## ✨ الميزات الإضافية

### حذف صور من الدفعة
```
1. مرر الماوس على الصورة في المعاينة
2. اضغط على × الذي يظهر
3. سيتم حذف الصورة من الدفعة
```

### معاينة سريعة
```
✓ عند رفع الصور، ستظهر معاينة مصغرة
✓ الصور الأمامية: رقم عادي (1, 2, 3...)
✓ الصور الخلفية: رقم مع B (B1, B2, B3...)
```
