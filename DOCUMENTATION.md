## المتطلبات والتثبيت

### المتطلبات الأساسية
- ✅ متصفح ويب حديث (Chrome, Firefox, Safari, Edge)
- ✅ 100MB مساحة حرة تقريباً في جهازك
- ✅ اتصال إنترنت (مرة واحدة فقط لتحميل المكتبات)
- ❌ **لا يتطلب** تثبيت برامج إضافية
- ❌ **لا يتطلب** خادم ويب

### التثبيت

#### الطريقة 1: الفتح المباشر (الأسهل)
```
1. انسخ مجلد 'card' إلى أي مقان على جهازك
2. انقر نقراً مزدوجاً على ملف 'index.html'
3. سيفتح تلقائياً في متصفحك الافتراضي
```

#### الطريقة 2: عبر خادم ويب محلي
```
إذا كنت تستخدم XAMPP أو محرك ويب آخر:
1. ضع المجلد في htdocs/
2. ادخل إلى: http://localhost/card/index.html
3. يجب أن تُشاهد الصفحة بنجاح
```

#### الطريقة 3: الفتح من المتصفح
```
1. افتح المتصفح
2. اضغط Ctrl+O
3. اختر ملف 'index.html'
4. اضغط موافق
```

## البدء السريع

### أول مشروع خاص بك (5 دقائق)

**الخطوة 1: رفع الصور**
```
1. اذهب إلى تبويب "الرفع 📤"
2. اختر حجم البطاقة (ISO 7810 للبطاقات القياسية)
3. اختر الصورة الأمامية من جهازك
4. اختر الصورة الخلفية من جهازك
```

**الخطوة 2: المعاينة**
```
- ستظهر الصورة الأمامية في المعاينة تلقائياً
- اضغط على "الخلف" في الإعدادات لرؤية الصورة الخلفية
```

**الخطوة 3: الضبط الأساسي**
```
في تبويب "الإعدادات ⚙️":
- استخدم منزلق التكبير لملاءمة الصورة بشكل صحيح
- استخدم الدوران إذا لزم الأمر
- استخدم الإزاحة لتحريك الصورة
```

**الخطوة 4: الطباعة أو التصدير**
```
اذهب إلى "التصدير 💾" واختر:
- "طباعة" للطباعة المباشرة
- "تصدير PDF" لحفظ ملف
- "تصدير PNG" لحفظ صورة
```

## شرح المميزات

### 1️⃣ تبويب الرفع (Upload)

#### حجم البطاقة
```
معايير متعددة:
- ISO 7810 (85.6×54mm): البطاقات الدولية ✓ موصى به
- بطاقة ائتمان (85×54mm): البطاقات المصرفية
- مخصص: حسب احتياجاتك الخاصة
```

#### وحدات القياس
```
اختر الوحدة المناسبة لك:
- ملليمتر (mm): الأدق والأكثر دقة
- بوصة (inch): الشائعة في الدول الغربية
- سنتيمتر (cm): اختيار عملي
```

#### رفع الصور
```
صيغ مدعومة:
- JPEG (.jpg, .jpeg)
- PNG (.png) ⭐ الأفضل
- GIF (.gif)
- WebP (.webp)

أحجام مدعومة:
- الحد الأدنى: 100×100 بكسل
- الحد الأقصى: 10 ميجابايت
```

### 2️⃣ تبويب الإعدادات (Settings)

#### اختيار الجانب
```
- الأمام (Front): الصورة الأمامية للبطاقة
- الخلف (Back): الصورة الخلفية للبطاقة

ملاحظة: يتم معالجة كل جانب بشكل مستقل
```

#### التحكم بالصورة

**التكبير/التصغير (Zoom)**
```
- نطاق: 50% إلى 300%
- الافتراضي: 100%
- الفائدة: ملاءمة الصورة داخل منطقة البطاقة
- مثال: إذا كانت صورتك كبيرة جداً، قلل الـ zoom
```

**الدوران (Rotation)**
```
- نطاق: 0° إلى 360°
- الزيادة: درجة واحدة
- الفائدة: تصحيح اتجاه الصورة
- مثال: إذا كانت الصورة مائلة، استخدم هذا
```

**الإزاحة الأفقية (Offset X)**
```
- نطاق: -50 إلى 50
- الفائدة: تحريك الصورة يميناً أو يساراً
- موجب: يحرك يميناً
- سالب: يحرك يساراً
```

**الإزاحة العمودية (Offset Y)**
```
- نطاق: -50 إلى 50
- الفائدة: تحريك الصورة لأعلى أو لأسفل
- موجب: يحرك لأسفل
- سالب: يحرك لأعلى
```

#### الخيارات المتقدمة

**الشبكة المساعدة** ☑️
```
فائدة: تساعدك على محاذاة الصور بدقة
- تعرض شبكة شفافة بفواصل 5 ملم
- تساعد في توزيع العناصر بتناسق
- يمكن تفعيلها/تعطيلها في أي وقت
```

**علامات التسجيل** ☑️
```
فائدة: توضح منطقة الطباعة الآمنة
- توضع في أركان البطاقة الأربعة
- تساعد في معايرة الطابعة
- مفيدة للطباعات الاحترافية
```

**عكس أفقياً** ☑️
```
فائدة: قلب الصورة أفقياً (انعكاس)
- بعض الطابعات تتطلب هذا
- يعكس الصورة من اليمين إلى اليسار
```

**عكس عمودياً** ☑️
```
فائدة: قلب الصورة عمودياً
- يقلب الصورة من الأعلى إلى الأسفل
```

### 3️⃣ تبويب النصوص (Text)

#### إضافة نص جديد
```
1. انقر على "+ إضافة حقل نص"
2. سيظهر حقل جديد
3. اكتب النص المطلوب
4. اختر الحجم واللون
5. سيظهر على البطاقة تلقائياً
```

#### تعديل النصوص
```
- النص: اكتب الكلمات المطلوبة
- الحجم: من 8 إلى 72 نقطة
- اللون: اختر من لوحة الألوان
```

#### حذف النصوص
```
- اختر النص وانقر "حذف"
- أو انقر "مسح جميع النصوص" لحذف الكل
```

### 4️⃣ تبويب التصدير (Export)

#### خيارات الطباعة

**عدد النسخ على الصفحة الواحدة**
```
- اختر 1-10 نسخ
- مفيد إذا أردت عدة بطاقات على صفحة A4
- سيتم ترتيبها تلقائياً
```

**إضافة حد الطباعة (Bleed)**
```
☑️ موصى به
- يضيف 3 ملم حول البطاقة
- يسمح بهامش أمان عند القص
- ينصح به للشركات والطباعات المهنية
```

**توسيط وقص**
```
☑️ اختياري
- يضمن توسيط البطاقة على الصفحة
- يسهل عملية القص اليدوي
```

#### خيارات التصدير

**تصدير PDF** 📄
```
الميزات:
- ملف واحد يحتوي على الجانبين
- جاهز للطباعة مباشرة
- محفوظ بدقة 300 DPI
- يحتفظ بالألوان الدقيقة

الاستخدام:
1. انقر "تصدير PDF"
2. اختر مقان الحفظ
3. سيبدأ التحميل تلقائياً
```

**تصدير PNG** 🖼️
```
الميزات:
- صورة عالية الجودة
- بدون فقدان للجودة
- مناسبة للعرض الرقمي
- يمكن تحريرها لاحقاً

صيغة الملف: استخدم للتحرير والعرض فقط
```

**طباعة** 🖨️
```
الخطوات:
1. انقر "طباعة"
2. ستفتح نافذة الطباعة
3. اختر الطابعة الخاصة بك
4. ⚠️ تأكد من تفعيل الخيارات أدناه
```

## الإعدادات المتقدمة

### إعدادات الطابعة (مهم جداً!)

للطباعة على جهتي الورقة بشكل صحيح، يجب أن:

```
1. فتح إعدادات الطابعة
   - Windows: لوحة التحكم → الأجهزة والطابعات
   - Mac: System Preferences → Printers & Scanners
   - Linux: نظام الطباعة الافتراضي

2. اختر "Duplex Printing" ✓
   - البحث عن "Double Sided" أو "Both Sides"
   - تأكد من تفعيله

3. اختر "Flip on Long Edge" ✓
   - الخيار الافتراضي والأفضل
   - يسمح بقلب الورقة على الحافة الطويلة

4. اضبط الهوامش على "None" ✓
   - ابحث عن "Margins" أو "Paper Margins"
   - اختر "No Margins"

5. اختر "Best Quality" ✓
   - جودة الطباعة الأعلى
   - استخدم "600 dpi" أو "Best"

6. اختر ورقك الصحيح ✓
   - احرص على اختيار الورق المناسب
   - (300+ gsm موصى به)
```

### حفظ القوالب (Templates)

#### الفائدة
```
توفير الوقت والجهد:
- احفظ إعداداتك بعد ضبطها
- استرجعها فوراً للمشاريع المشابهة
- تجنب إعادة الضبط كل مرة
```

#### الحفظ
```
1. بعد تخصيص البطاقة بالكامل
2. اذهب إلى الإعدادات وتأكد من رضاك
3. انقر "حفظ القالب 💾"
4. أدخل اسماً للقالب (مثال: "بطاقات المؤتمر 2026")
5. سيتم حفظه محلياً في جهازك
```

#### الاسترجاع
```
1. انقر "تحميل القالب 📂"
2. ستظهر قائمة بالقوالب المحفوظة
3. انقر على القالب المطلوب
4. سيتم استرجاع جميع الإعدادات تلقائياً
```

#### الحذف
```
1. اضغط على "تحميل القالب"
2. أنقر حذف بجانب القالب المطلوب
3. سيتم حذفه نهائياً (أكد أولاً)
```

## نصائح للحصول على أفضل النتائج

### 1. جودة الصور 📸

**اختيار الصور**
```
✓ استخدم صور بدقة عالية (300 DPI+)
✓ تأكد من أن وضوح الصورة ممتاز
✓ تجنب الصور الضبابية أو الشديدة البكسل
✓ تحقق من الإضاءة والألوان قبل الرفع
```

**حجم الملف**
```
✓ الحد الأمثل: 2-5 ميجابايت
✓ الحد الأقصى: 10 ميجابايت
✓ الصيغ الموصى بها: PNG، JPEG بجودة عالية
```

### 2. المحاذاة الدقيقة 🎯

**استخدام الشبكة المساعدة**
```
1. فعّل "عرض الشبكة 🔲"
2. ستظهر خطوط بفواصل 5 ملم
3. استخدمها كدليل لمحاذاة الصور
4. يمكنك تعطيلها قبل الطباعة النهائية
```

**الطباعة التجريبية**
```
1. اطبع نسخة اختبار على ورقة عادية أولاً
2. تحقق من مواضع الصور والنصوص
3. قارن مع الأصلية المطلوبة
4. اضبط الإزاحة إذا لزم الأمر
5. اطبع النسخة النهائية بثقة
```

### 3. الألوان والطباعة 🎨

**إعدادات الألوان**
```
✓ استخدم وضع "Satin" للوان غنية
✓ استخدم "Best Quality" للألوان الدقيقة
✓ اختبر على عينة صغيرة أولاً
✓ قد تختلف الألوان قليلاً عن الشاشة
```

**الطابعة الصحيحة**
```
✓ استخدم طابعة حديثة تدعم Duplex
✓ تأكد من كفاية الحبر/الركيزة
✓ نظف رؤوس الطباعة إذا لزم الأمر
✓ استخدم إعدادات الطابعة الأمثل
```

### 4. نوع الورق 📄

**الخيارات الموصى بها**
```
① ورق محترف (Best):
   - 280-300 gsm matte/silk
   - مثالي للبطاقات الاحترافية
   - تكلفة عالية نسبياً

② ورق عالي الجودة (Good):
   - 220-250 gsm glossy/matte
   - توازن جيد بين الجودة والسعر
   - مناسب لأغلب الاستخدامات

③ ورق عادي (Average):
   - 150-200 gsm
   - مناسب للاستخدام الداخلي
   - اقتصادي وسهل التوفر
```

**الحفظ الصحيح**
```
✓ احفظ الورق في مكان جاف
✓ تجنب الرطوبة والحرارة الشديدة
✓ حافظ على المغلف الأصلي
✓ لا تعرضه لأشعة الشمس المباشرة
```

### 5. عملية القص ✂️

**الأدوات المطلوبة**
```
① مقص ورق احترافي (أفضل)
   - قطع مستقيم ودقيق
   - قليل الجهد المطلوب

② سكين حادة مع مسطرة
   - اطلب من محترف إذا لم تكن متأكد
   - احذر من الإصابة

③ ماكينة قطع (الأفضل للكميات الكبيرة)
   - نتائج احترافية
   - توفر الوقت والجهد
```

**خطوات القطع الآمن**
```
1. حدد خطوط القطع بقلم خفيف
2. استخدم مسطرة معدنية كدليل
3. أمسك الورقة بثبات
4. قص بحركة واحدة سلسة
5. لا تحاول إعادة القص إذا كان غير مستقيم
    (قد يتضرر الحافة)
```

## استكشاف الأخطاء

### مشكلة: الصور لا تظهر بعد الرفع 🖼️

**الأسباب والحلول**
```
السبب 1: صيغة الملف غير مدعومة
الحل:
- تأكد من استخدام JPG أو PNG
- حول صورك باستخدام Photoshop أو Paint

السبب 2: حجم الملف كبير جداً
الحل:
- استخدم أداة ضغط الصور
- قلل دقة الصورة إلى 300 DPI
- استخدم موقع TinyPNG.com

السبب 3: مشكلة في المتصفح
الحل:
- أعد تحميل الصفحة (F5)
- امسح ذاكرة التخزين المؤقت (Ctrl+Shift+Delete)
- جرب متصفح آخر

السبب 4: مشكلة في جهازك
الحل:
- تأكد من وجود مساحة حرة كافية
- أعد تشغيل المتصفح
- أعد تشغيل جهازك
```

### مشكلة: المحاذاة غير صحيحة عند الطباعة 📐

**الأسباب والحلول**
```
السبب 1: إعدادات الطابعة خاطئة
الحل:
- تأكد من تفعيل Duplex Printing
- تأكد من اختيار "Flip on Long Edge"
- اضبط الهوامش على "None"

السبب 2: الصورة نفسها غير محاذاة
الحل:
- استخدم الشبكة المساعدة
- اضبط الإزاحة (Offset)
- استخدم المنزلقات بدقة

السبب 3: الورقة غير موضوعة بشكل صحيح
الحل:
- تأكد من وضع الورقة بشكل صحيح
- استخدم دليل الورق من الطابعة
- اطبع نموذج اختبار قبل الطباعة النهائية

السبب 4: الطابعة نفسها بها مشكلة
الحل:
- نظف رؤوس الطباعة
- تحقق من صيانة الطابعة
- جرب طابعة أخرى
```

### مشكلة: الطباعة على جهة واحدة فقط 📄

**الأسباب والحلول**
```
السبب 1: Duplex Printing معطل
الحل:
1. افتح إعدادات الطابعة
2. ابحث عن "Printing" أو "Print Settings"
3. فعّل "Duplex Printing" أو "Two-Sided"
4. اختر "Flip on Long Edge"

السبب 2: الطابعة لا تدعم Duplex
الحل:
- اطبع الجانب الأول
- أدخل الورق مجدداً من الأعلى
- اطبع الجانب الثاني يدوياً
(سيتطلب هذا دقة في المحاذاة)

السبب 3: حدث خطأ في الطابعة
الحل:
- أعد تشغيل الطابعة
- تحقق من وجود رسائل خطأ
- استدعِ الدعم الفني
```

### مشكلة: ملف PDF لا يُفتح 📄

**الأسباب والحلول**
```
السبب 1: برنامج قراءة PDF غير مثبت
الحل:
- حمّل Adobe Reader من adobe.com
- أو استخدم متصفح متوافق

السبب 2: الملف معطوب
الحل:
- حاول التصدير مجدداً
- جرب متصفح مختلف
- تحقق من المساحة الحرة

السبب 3: مشكلة في المتصفح
الحل:
- أعد تحميل الصفحة
- امسح ذاكرة التخزين المؤقت
- جرب متصفح آخر

السبب 4: ملف غير مكتمل
الحل:
- تأكد من انتهاء التحميل كاملاً
- تحقق من شريط التحميل في المتصفح
- انتظر بضع ثواني قبل الفتح
```

### مشكلة: النصوص لا تظهر على البطاقة ✏️

**الأسباب والحلول**
```
السبب 1: النص باللون الأبيض على خلفية بيضاء
الحل:
- اختر لون مختلف للنص
- استخدم الألوان الداكنة

السبب 2: حجم النص صغير جداً
الحل:
- زيادة حجم النص إلى 14+
- استخدم خطوط واضحة

السبب 3: النص خارج نطاق البطاقة
الحل:
- استخدم الشبكة المساعدة
- اضبط موضع النص يدوياً
- استخدم مربع النص الصحيح

السبب 4: قد لم يتم حفظ النص
الحل:
- تأكد من الضغط على "Add"
- تحقق من ظهور النص في القائمة
- جرب إضافة النص مجدداً
```

### مشكلة: القالب المحفوظ لا يُحمّل 💾

**الأسباب والحلول**
```
السبب 1: Local Storage معطل
الحل:
- تحقق من إعدادات الخصوصية
- فعّل حفظ البيانات المحلية
- تأكد من عدم حذف بيانات المتصفح

السبب 2: القالب قديم جداً
الحل:
- قد يكون البرنامج قد تحدّث
- أعد إنشاء القالب من جديد
- لا تحتفظ بقوالب قديمة جداً

السبب 3: مشكلة في المتصفح
```
